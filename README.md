# Harshini Sanjana - Personal Portfolio

A modern, cosmic-themed personal portfolio built with React, Vite, and Tailwind CSS.

## 🚀 Live Demo
Check out the live site: [https://harshinisanjana.github.io/personal-portfolio/](https://harshinisanjana.github.io/personal-portfolio/)

## ✨ Features
- **Cosmic Design**: High-fidelity dark mode with floating particles and nebula glows.
- **Responsive Layout**: Fully optimized for mobile and desktop viewing.
- **Tech Stack**: Showcases skills in Python, Java, React, SQL, and more.
- **Automated Deployment**: Integrated with GitHub Pages for seamless updates.

## 🛠️ Built With
- **Frontend**: React.js, Tailwind CSS
- **Icons**: Lucide React, React Icons
- **Animation**: CSS Keyframes, Tailwind Utilities
- **Deployment**: Gh-pages
